===========================================
HOW TO ADD YOUR PHOTOS
===========================================

1. LANDING/HERO SECTION PHOTO:
   - File name: hero-photo.jpg
   - Location: Put it in this "images" folder
   - Recommended size: 1920x1080 pixels or larger (wide/landscape)
   - Format: JPG, PNG, or WebP

2. ABOUT SECTION PHOTO:
   - File name: about-photo.jpg
   - Location: Put it in this "images" folder
   - Recommended size: 1000x800 pixels or larger (landscape)
   - Format: JPG, PNG, or WebP

===========================================
STEPS:
===========================================

1. Copy your photos to this "images" folder
2. Rename them to:
   - hero-photo.jpg (for landing section)
   - about-photo.jpg (for about section)

3. Make sure the file names match exactly:
   - hero-photo.jpg
   - about-photo.jpg

4. Refresh your browser to see the changes!

===========================================
NOTE:
===========================================

- You can use JPG, PNG, or WebP format
- If your files have different names, you can rename them
- Or update the file names in index.html (lines 1131 and 1185)
